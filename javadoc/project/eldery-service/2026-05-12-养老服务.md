# 智慧医养云平台 - 技术说明文档

> **项目名称：** pro-older-service（基于 RuoYi-Vue-Plus 3.5.0）
> **最后更新：** 2026-05-12
> **项目版本：** 3.5.0

---

## 一、项目概述

智慧医养云平台是一套面向养老机构的综合管理系统，涵盖老人信息管理、护理服务、费用结算、智能设备对接、健康监测、统计分析等核心业务功能。

**技术选型核心依据：**
- 基于 RuoYi-Vue-Plus 3.5.0 增强版框架
- 采用前后端分离架构（Vue.js + Spring Boot）
- 多数据源架构（MySQL + TDengine 时序数据）
- 微服务友好的模块化设计（60+ 业务模块）

## 六、安全与认证

### 6.1 认证方案

采用 **Sa-Token** 实现无状态 JWT 认证：
- Token 有效期：**24 小时**
- 支持单点登录（SSO）
- 支持 token 续期机制
- 路径排除配置（白名单）：
  - `/login`、`/register`、`/captchaImage`
  - `/mp/**`（小程序接口）
  - `/gzh/**`（公众号接口）
  - `/staff/**`（员工 App 接口）
  - `/webjars/**`、`/swagger-resources/**`（API 文档）

### 6.2 安全防护

| 防护措施 | 实现方式 |
|---------|---------|
| XSS 过滤 | Pattern 方式过滤请求参数 |
| SQL 注入防护 | MyBatis-Plus 参数绑定 + Druid 防火墙 |
| 数据权限 | AOP 切面自动注入数据范围过滤 |
| 操作日志 | 切面记录所有变更操作 |
| 验证码 | 支持算数/圆圈两种类型 |
