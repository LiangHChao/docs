# 居家医护服务系统 - 技术说明文档

## 1. 项目概述

**项目名称**: 居家医护服务系统 (Home Care System)
**基础框架**: RuoYi v3.9.0
**系统类型**: 互联网医院 + 医护上门服务平台
**项目版本**: 3.9.0

### 1.1 主要功能模块

| 模块 | 说明 |
|------|------|
| 医护上门 | 护士到家服务预约与管理 |
| 在线问诊 | 患者与医生在线沟通、图文/视频问诊 |
| 电子处方 | 医生开具电子处方及药店配送 |
| 就诊卡 | 会员卡管理、激活码、权益服务 |
| 支付系统 | 微信支付、支付宝集成 |
| 医院信息 | 科室、医生、排班等基础数据 |
| 商品商城 | 医疗商品销售与售后服务 |
| 小程序 | 微信小程序用户端 |

## 3. 项目结构

```

├── home-care/                    # 后端项目
│   ├── ruoyi-admin/              # 主应用入口、控制器

│   ├── ruoyi-common/             # 通用工具类

│   ├── ruoyi-quartz/             # 定时任务模块

│   ├── ruoyi-payment/           # 支付模块
│   ├── ruoyi-medical-card/      # 就诊卡模块
│   ├── ruoyi-consultation/       # 问诊模块
│   ├── ruoyi-prescription/       # 处方模块
│   ├── ruoyi-nurse-care/         # 医护上门模块
│   ├── ruoyi-hospital/           # 医院信息模块
│   ├── ruoyi-shop/               # 商城模块
```

## 7. 构建与部署

### 7.1 后端构建

```bash
# 进入后端目录
cd home-care

# 编译打包（跳过测试）
mvn clean package -DskipTests

# 运行主应用
java -jar ruoyi-admin/target/ruoyi-admin.jar
```

### 7.2 前端构建

**管理后台**:
```bash
cd home-care/ruoyi-ui

# 安装依赖
pnpm install

# 开发模式
pnpm run dev

# 生产构建
pnpm run build:prod
```

**微信小程序**:
```bash
cd home-care-weapp

# 安装依赖
pnpm install

# 微信小程序开发
pnpm run dev:weapp

# 微信小程序构建
pnpm run build:weapp
```

## 8. 开发规范

### 8.1 代码生成

项目集成了RuoYi代码生成器，可通过管理后台访问：
- 路径: 系统管理 > 代码生成
- 支持生成: 实体、Mapper、Service、Controller、Vue页面

### 8.2 实体类规范

```
domain/
├── entity/     # 数据库实体
├── bo/         # 业务请求对象
└── vo/         # 视图响应对象
```

### 8.3 命名规范

| 类型 | 规范 |
|------|------|
| 类名 | 大驼峰命名 (如: UserService) |
| 方法名 | 小驼峰命名 (如: getUserById) |
| 包名 | 全小写 (如: com.ruoyi.common) |
| 常量 | 全大写下划线分隔 (如: MAX_RETRY_COUNT) |
