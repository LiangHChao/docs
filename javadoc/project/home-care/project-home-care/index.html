居家医护服务系统 - 技术说明文档

1. 项目概述​

项目名称: 居家医护服务系统 (Home Care System) 基础框架: RuoYi v3.9.0 系统类型: 互联网医院 + 医护上门服务平台 项目版本: 3.9.0

1.1 主要功能模块​

模块说明
医护上门护士到家服务预约与管理
在线问诊患者与医生在线沟通、图文/视频问诊
电子处方医生开具电子处方及药店配送
就诊卡会员卡管理、激活码、权益服务
支付系统微信支付、支付宝集成
医院信息科室、医生、排班等基础数据
商品商城医疗商品销售与售后服务
小程序微信小程序用户端

2. 技术架构​

2.1 后端技术栈​

技术版本说明
Java17开发语言
Spring Boot2.7.18核心框架
Spring Cloud2021.0.9微服务框架 (已禁用)
Alibaba Cloud2021.0.6.0阿里云SDK
MyBatis Plus3.5.1ORM框架
Druid1.2.23数据库连接池
Redis-缓存 (Lettuce客户端)
Nacos-服务发现 (默认禁用)

管理后台 (ruoyi-ui):

技术版本说明

Vite6.3.5构建工具

Axios1.9.0HTTP客户端

微信小程序 (home-care-weapp):

技术版本说明

React18.0.0UI库
NutUI3.0.19京东风格的组件库
Zustand5.0.8状态管理
TailwindCSS3.4.18CSS框架

2.3 中间件与服务​

服务说明
MySQL关系型数据库
Redis缓存、会话存储

腾讯云SMS短信服务

支付宝支付接口

3. 项目结构​

3.1 目录结构​

7. 构建与部署​

7.1 后端构建​

# 进入后端目录
cd home-care

# 编译打包（跳过测试）
mvn clean package -DskipTests

# 运行主应用
java -jar ruoyi-admin/target/ruoyi-admin.jar

7.2 前端构建​

管理后台:

cd home-care/ruoyi-ui

# 安装依赖
pnpm install

# 开发模式
pnpm run dev

# 生产构建
pnpm run build:prod

微信小程序:

cd home-care-weapp

# 安装依赖
pnpm install

# 微信小程序开发
pnpm run dev:weapp

# 微信小程序构建
pnpm run build:weapp

8. 开发规范​

8.1 代码生成​

项目集成了RuoYi代码生成器，可通过管理后台访问：

  • 路径: 系统管理 > 代码生成
  • 支持生成: 实体、Mapper、Service、Controller、Vue页面

8.2 实体类规范​

domain/
├── entity/     # 数据库实体
├── bo/         # 业务请求对象
└── vo/         # 视图响应对象

8.3 命名规范​

类型规范
类名大驼峰命名 (如: UserService)
方法名小驼峰命名 (如: getUserById)
包名全小写 (如: com.ruoyi.common)
常量全大写下划线分隔 (如: MAX_RETRY_COUNT)
